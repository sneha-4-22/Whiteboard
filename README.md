# 🎨 Real-Time Whiteboard Collaboration Platform

A modern, collaborative whiteboard platform enabling seamless, real-time multi-user editing and drawing. Built with React and a Node.js (Express) backend, this solution empowers teams and classrooms to brainstorm, sketch, and collaborate visually—anytime, anywhere—with sub-second latency.

## ✨ Key Features

- <b>Real-Time Collaboration:</b> Draw and edit together with others, powered by <b>Socket.IO</b> for instant, synchronized whiteboard updates.
- <b>Secure Authentication:</b> <b>JWT</b>-based authentication ensures safe and private user sessions.
- <b>Persistent Storage:</b> <b>MongoDB</b>-backed session saving and restoration, with optimized queries and indexing for 30% faster API response times.
- <b>RESTful API:</b> Clean, well-structured CRUD APIs for managing users, sessions and whiteboard data.
- <b>Consistent State:</b> Robust real-time state management keeps all users in sync, always seeing the latest changes.

## 🛠️ Tech Stack

- <b>Frontend:</b> React
- <b>Backend:</b> Node.js & Express
- <b>Database:</b> MongoDB
- <b>Real-Time:</b> Socket.IO
- <b>Authentication:</b> JWT
